Extraer links de un archivo dlc

A. Gómez - When I was a young
2009-07-15
Software
Jdownloader, dlc, extractor

Algunas personas que usen la descarga directa como alternativa preferida a la hora de obtener contenidos desde la red, sabrán que en algunas páginas tienen la insana costumbre de poner los enlaces codificados dentro de un archivo .dlc que sólo puede usarse con el gestor de descargas JDownloader. Si eres de esos que pasan de usar un gestor basado en Java como JDownloader, es muy probable que te hayas planteado cómo se podría tratar de extraer los links de esos archivos para obtener los links directos y así poder usar cualquier otro gestor como el Megamanager, entonces, aquí tienes la solución. Buscando por la red, he llegado a una web llamada Containerex.info que ofrece un programa muy sencillo con el cual podréis extraer los links de los archivos .dlc. Para ello sólo tendréis que:

  1. Descargar el programa desde la página web oficial
  2. Descomprimirlo usando cualquier descompresor, como Winzip, Winrar. De modo que obtendrás un ejecutable.
  3. Ejecutamos el archivo y lo primero que veremos será una pantalla de bienvenida y posteriormente una de publicidad, clickamos en cerrar en esta última.
  4. Veremos la interfaz principal, en ella, pulsaremos en Open y seleccionaremos el archivo .dlc que queremos extraer.
  5. Voila! Ya tendremos una lista con los links que contiene el archivo .dlc

Os recuerdo que muchos uploaders usan este sistema para evitar que los links sean públicos y así las subidas duren más, de modo que os pido que hagáis un uso responsable del programa.
