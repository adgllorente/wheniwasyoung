Ftp2Share.com, otra facilidad a la hora de compartir

A. Gómez - When I was a young
2007-09-14
Webs interesantes

Hace un tiempo comentaba la existencia de MassMirror.com una web que facilitaba la subida de archivos a servidores externos para luego poder descargar directamente de ellos.

Ahora es el turno de Ftp2Share.com, una web en la que subes el archivo que deseas compartir a sus servidores y ellos se encargan de distribuirlo por Rapidshare, Megaupload y Uploading entre otros.

Una vez subido sólo tienes que compartir el link que te da Ftp2Share con tus amigos y ellos se encargan de mostrarte los links a los servidores que todavía contengan el archivo.

Lo bueno de esta web es que comprueban los links existentes para que puedes alternar entre ellos y así terminar con los posibles limites de descarga que alguno de estos servidores tienen.

En la imagen de la izquierda podéis ver de una forma gráfica como funciona este servicio.
